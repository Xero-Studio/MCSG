# 📦 快速开始

**系统要求：**
- Python 3.6 或更高版本
- Java（用于运行Minecraft服务器）

**安装步骤：**
1. 下载或克隆此仓库
2. 确保在同一目录下有Minecraft服务器jar文件（例如 `server.jar`）
3. 运行程序：
   - **Windows**: 双击 `start.bat` 或运行 `python start.py`
   - **Linux/macOS**: 运行 `./start.sh` 或 `python3 start.py`

程序会自动创建配置文件并处理EULA协议。你只需要有一个服务器核心文件就可以开始使用了。
